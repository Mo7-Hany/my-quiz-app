
# 🤖 AI-Powered Question & Quiz Generator

A smart web application built with **Python** and **Streamlit** that leverages the **Google Gemini API** to generate high-quality educational questions and interactive quizzes instantly.

---

## 🚀 Live Demo
Experience the application live here: [**https://mainpy-lgmktmjhfsepgauvtdjv7t.streamlit.app/**]

---

## ✨ Features
* **AI-Driven Content:** Uses Google Gemini Pro to generate accurate and context-aware questions.
* **Customizable Quizzes:** Users can specify topics, difficulty levels, or types of questions.
* **Secure Implementation:** Integrated with **Streamlit Secrets** to ensure API keys are never exposed.
* **Fast & Responsive:** Clean UI designed for a smooth user experience.
* **Rate-Limit Handling:** Built-in logic to manage API quotas effectively.

---

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Framework:** [Streamlit](https://streamlit.io/)
* **AI Engine:** [Google Generative AI (Gemini)](https://ai.google.dev/)
* **Deployment:** Streamlit Cloud
